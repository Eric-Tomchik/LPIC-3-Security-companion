# LPIC-3 Security (303-300) — Online Companion

Free online companion resources for the **LPIC-3 Security Complete Study Guide** (Exam 303-300, Version 3.0) by Eric Tomchik, published by ArcLight Press.

## 🔗 Live Site

**[https://eric-tomchik.github.io/LPIC-3-Security-companion/](https://eric-tomchik.github.io/LPIC-3-Security-companion/)**

## 📚 What's Inside

| Section | Description |
|---------|-------------|
| **Chapters** | Learning objectives and section breakdowns for all 16 chapters across 5 exam topics |
| **Command Reference** | 9 quick-reference cards covering OpenSSL, DNSSEC, LUKS, SELinux, AppArmor, iptables, nftables, Nmap, and more |
| **Interactive Labs** | 20+ hands-on labs with browser-based terminal simulators — zero setup required |
| **Glossary** | 73 searchable key terms with definitions |
| **Flashcards** | 90+ interactive flashcards with 11 category filters, keyboard shortcuts, and shuffle |
| **Practice Quiz** | 65 exam-style questions (multiple choice + fill-in-the-blank) with instant feedback |
| **Errata** | Corrections and updates |
