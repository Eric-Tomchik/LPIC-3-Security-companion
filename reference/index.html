🔑 Cryptography & Certificates

TaskCommand
Generate RSA keyopenssl genrsa -aes256 -out key.pem 4096
Generate key (modern)openssl genpkey -algorithm RSA -pkeyopt rsa_keygen_bits:4096 -out key.pem
Generate Ed25519 SSH keyssh-keygen -t ed25519
Create CSRopenssl req -new -key key.pem -out csr.pem
Sign certificateopenssl x509 -req -in csr.pem -CA ca.crt -CAkey ca.key -out cert.pem
Inspect certificateopenssl x509 -in cert.pem -text -noout
Test TLS connectionopenssl s_client -connect host:443
Show full cert chainopenssl s_client -connect host:443 -showcerts
Get Let's Encrypt certcertbot certonly --standalone -d example.com
Renew all certscertbot renew
PEM → DERopenssl x509 -in cert.pem -outform DER -out cert.der
Create PKCS#12 bundleopenssl pkcs12 -export -in cert.pem -inkey key.pem -out bundle.p12
Verify cert matches keyopenssl x509 -noout -modulus -in cert.pem | md5sum
